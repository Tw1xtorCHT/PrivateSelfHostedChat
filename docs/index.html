PrivateSelfHostedChat

Приватный самохостируемый чат с голосовыми звонками. Без регистрации, без логов, без следов. Разворачивается на любом VPS за 3 минуты одной командой.

💬 Анонимный чат

Без регистрации и аккаунтов. Зашёл по ссылке, ввёл пароль, общаешься. Закрыл вкладку — всё исчезло.

📞 P2P звонки

Голосовые звонки через WebRTC. Трафик идёт напрямую между браузерами. Сервер не слышит разговор.

🔒 Автосмена пароля

Пароль меняется автоматически каждый час. Новый пароль приходит в Telegram. Старые сессии уничтожаются.

🤖 Telegram бот

Управление чатом через бота. Одна кнопка — сброс чата, смена пароля, уведомление в Telegram.

🗃 Передача файлов

PDF, фото, документы до 50 МБ. Перетащи файл в окно чата и отправь. После очистки — стёрто навсегда.

🔓 HTTPS

Автоматический SSL сертификат через Let's Encrypt. Весь трафик зашифрован. Уровень защиты как в Telegram.

Установка за 3 минуты

Шаг 1 — Создайте Telegram бота

Откройте @BotFather в Telegram, отправьте /newbot и скопируйте токен.
Узнайте свой Chat ID у @userinfobot.

Шаг 2 — Получите бесплатный домен

Зайдите на duckdns.org, создайте поддомен и укажите IP вашего сервера.

Шаг 3 — Запустите установку

Подключитесь к VPS по SSH и выполните команду:

curl -fsSL https://raw.githubusercontent.com/Tw1xtorCHT/PrivateSelfHostedChat/main/install.sh | bash

Скрипт спросит токен бота, Chat ID и домен. Всё остальное установится автоматически.

Как это работает

Браузер → HTTPS (Caddy) → Auth (Token) → Chat (m1k1o/chat)
Звонки → Signal Server → P2P (WebRTC) → Напрямую между браузерами
Управление → Telegram Bot → Автосброс каждый час

Безопасность

✅ Сообщения только в RAM Ничего не пишется на диск. При перезапуске контейнера всё стирается безвозвратно.
✅ Открытый код Весь код доступен на GitHub. Без бэкдоров, без аналитики, без трекеров.
✅ Свой сервер Данные только на вашем VPS. Никаких третьих лиц, облаков и корпораций.
✅ P2P звонки Аудио идёт напрямую между устройствами. Сервер только сводит участников.
